MS Internet Explorer Version 5
Dial-up Modem Connection Local Area Network (LAN) Connection
  1. Select: Tools
  2. Select: Internet Options...
  3. Select: Connections
  4. In the Dial-up settings box find the icon with the name of your dial-up service. Click on the icon once to highlight it.
  5. Select: Settings...
  6. Check the box next to: "Automatically detect settings"
  7. Check the box next to: "Use automatic configuration script"
  8. enter this Address http://172.16.11.225:8080/proxy.pac
  9. click on OK
  10. click on OK on the Internet Options menu
